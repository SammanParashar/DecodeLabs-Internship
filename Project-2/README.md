# Project 2 - AWS EC2 Web Server

## Objective
Launch an Amazon EC2 instance, install and configure the Nginx web server, and access the webpage using the EC2 Public IP.

## AWS Services Used
- Amazon EC2
- Nginx

## Instance Details
- Instance Type: t3.micro (or t2.micro if used)
- Operating System: Amazon Linux
- Region: us-east-1

## Steps Performed
1. Launched an Amazon EC2 instance.
2. Connected to the instance using SSH.
3. Updated the system packages.
4. Installed the Nginx web server.
5. Started and enabled the Nginx service.
6. Allowed HTTP (Port 80) in the Security Group.
7. Accessed the default Nginx webpage using the EC2 Public IP.

## Result
Successfully deployed and accessed the Nginx web server through the EC2 Public IP.

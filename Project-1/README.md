# Project 1 - AWS S3 Static Website Hosting

## Objective
Host a static portfolio website using Amazon S3 and make it publicly accessible.

## AWS Services Used
- Amazon S3

## Steps Performed
1. Created an Amazon S3 bucket.
2. Uploaded the website files (HTML, CSS, and other assets).
3. Enabled Static Website Hosting.
4. Configured bucket permissions for public access.
5. Generated the S3 Website Endpoint.
6. Verified that the website was accessible through the S3 URL.

## Technologies Used
- HTML
- CSS
- Amazon S3

## Result
Successfully hosted a static portfolio website on Amazon S3 and accessed it using the S3 Website Endpoint.

# Project 3 - AWS RDS MySQL Database

## Objective
Create and configure an Amazon RDS MySQL database instance and make it available for database applications.

## AWS Services Used
- Amazon RDS
- MySQL

## Database Details
- Database Name: project3-db
- Engine: MySQL Community
- Instance Type: db.t3.micro
- Region: us-east-1
- Port: 3306
- Status: Available

## Steps Performed
1. Created an Amazon RDS MySQL database instance.
2. Configured database settings and storage.
3. Selected the db.t3.micro instance type.
4. Configured networking and security settings.
5. Waited for the database status to become **Available**.
6. Verified that the database instance was running successfully.

## Result
Successfully created and deployed an Amazon RDS MySQL database instance on AWS.
